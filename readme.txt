== Description ==
 
View and compare the highest odds on multiple sports, leagues and events. Shows matches from a selected league with the corresponding odds for the different teams.

We have odds on all the big sports and leagues. If you prefer to show odds from different betting companies on a single game, you can do that as well.

== Installation ==
 
1. Install the plugin
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The odds comparison widget will now be visible and you can place it where you want to.

== How to use ==
 
1. Get the relative url from Odds.nu webpage and copy/paste the relative url, i.e "hockey/nordamerika/nhl".
1. Choose maximum number of events/matches that you want to show in the widget.
1. Specify the height of your widget (in pixels).
